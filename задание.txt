1. Реализовать функцию Calculate класса WorkDayCalculator




2. На вход функции подается Дата отсчета, продолжительность 
в днях (целое число) и выходные 
(массив объектов с полями дата начала и дата конца, 
если дата начала совпадает с дата конца - значит выходной 
только этот день, 
если не совпадает - значит этот период выходной, 
например  23/04/2011 - 25/04/2011 - два выходных 
(Суббота + Воскресенье) и 25е - праздник,
 итого три выходных, может быть пустым). 
Массив отсортирован по дате начала, периоды не пересекаются.
  
3. На выходе функция должна вернуть Дату, которая 
высчитывается как дата начала + продолжительность 
без учета выходных.




4. Реализовать методы тестирования в классе 
WorkDayCalculatorTests по примеру с проверкой всех 
возможных граничных случаев, 
которые удастся придумать
